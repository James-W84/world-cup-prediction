# web/.ai.md

## 1. Local Feature Domain
- **Scope:** Next.js App Router UI for home, auth callback, matches, predictions, leagues.
- **Core Dependencies:** React 19, Next 15, Zustand, `src/lib/api.ts`, global CSS variables.

## 2. Invariants & Business Logic Rules
- **State Invariants:** Client pages using hooks must start with `'use client'`.
- **State Invariants:** Auth gate reads `useAuth`; unauthenticated initialized users redirect to `/`.
- **Data Mutations:** API calls include credentials; in production `getApiUrl()` returns `/api` (proxied via Next.js rewrite to `API_INTERNAL_URL`); in dev falls back to `NEXT_PUBLIC_API_URL` or `http://localhost:4000`.
- **Data Mutations:** Shared API types live in `src/lib/api.ts`; update consumers together.
- **Edge Cases:** Fetch helper returns `body.data`; endpoints with extra top-level fields need custom wrappers (`getMatches` returns `knockoutLocked`).

## 3. Testing & Mocking Constraints
- **Test File Location:** No test harness present. Add runner before `*.test.tsx`.
- **Mocks:** Mock `src/lib/api.ts` for page tests; mock Zustand stores for auth/prediction flows.
